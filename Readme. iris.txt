## Objective
Build a logistic regression model to classify iris flower species (Setosa, Versicolor, Virginica) using features like petal length, sepal width, etc.

## Dataset
[Iris Dataset - UCI Repository](https://archive.ics.uci.edu/ml/datasets/iris)

## Tools & Libraries
- Python
- pandas, numpy
- scikit-learn
- matplotlib, seaborn

## Steps Performed
- Data Exploration
- Checking for Missing Values
- Label Encoding of Target (`species`)
- Feature Scaling using StandardScaler
- Train-Test Split
- Logistic Regression Model Training
- Evaluation (Accuracy, Classification Report, Confusion Matrix, Cross-Validation)

## Accuracy
Achieved Accuracy: ~97%
